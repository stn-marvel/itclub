About IT Club

IT Club is a place for students to explore technology through hands-on learning, collaboration, creative projects, and competitions. Members are encouraged to develop practical skills while working together in a fun and supportive environment.

Programming

Learn to code in multiple languages and build software from scratch.

Web Development

Design and build modern websites and web applications.

Artificial Intelligence

Explore machine learning, neural networks, and AI-powered projects.

Cybersecurity

Understand security fundamentals and ethical hacking techniques.

Hardware Development

Learn Arduino, ESP32, sensors, IoT, and embedded systems.

UI/UX Design

Create intuitive interfaces and delightful user experiences.

What We Do

From learning the basics to building real projects, our activities are designed to be engaging and practical.

Coding Class

Learn HTML, CSS, JavaScript, Python, C++, and other programming fundamentals.

Hardware Workshop

Build electronic projects using Arduino, ESP32, sensors, and simple IoT systems.

Competition

Participate in technology competitions, hackathons, and school events.

Project Development

Create real-world projects such as websites, IoT devices, automation systems, or simple applications.
